# Домашнее задание к занятию «Системы контроля версий» - Петр Петров
### Задание 1. Создать и настроить репозиторий для дальнейшей работы на курсе
Благодаря добавленным файлам `.gitignore` будут игнорироваться:
**В корне проекта:**
- временные файлы;
- файлы кеша;
- логи-файлы;
- любые другие служебные файлы, не относящиеся к коду проекта.

**В каталоге `terraform/`:**
- директории `.terraform` (локальные данные провайдера);
- файлы состояния Terraform (`*.tfstate`, `*.tfstate.*`);
- логи падений (`crash.log`);
- файлы переменных (`*.tfvars`);
- файлы переопределений (`*override.tf`, `*override.tf.json`);
- конфигурационные файлы CLI (`terraform.rc`, `.terraformrc`).

Это позволяет не засорять репозиторий временными и генерируемыми файлами.
